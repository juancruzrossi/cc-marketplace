# cc-marketplace

Plugin marketplace for Claude Code by [@juancruzrossi](https://github.com/juancruzrossi).

## Usage

Register the marketplace in Claude Code:

```
/plugin marketplace add juancruzrossi/cc-marketplace
```

Then install any plugin:

```
/plugin install bridge@cc-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [bridge](https://github.com/juancruzrossi/sdd-bridge) | Spec-Driven Development framework — interview, plan, execute, review, debug |
